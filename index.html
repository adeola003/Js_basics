Variables

Variables are the basics elements manipulated in javascript.

They are four ways to declare variables in Javascript

  • Using "var"
  • Using "let"
  • Using "const"
  • Using nothing
Variables are container to store data values. They can be made of "string", "numbers", "objets", "arrays" and so on.

Arrays

In JavaScript, the array is a single variable that is used to store different elements. It is often used when we want
to store a list of elements and access them by a single variable. Unlike most languages where the array is a reference
to the multiple variables, in JavaScript array is a single variable that stores multiple elements.
